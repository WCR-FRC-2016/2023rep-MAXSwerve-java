# Java analysis

## Pros

- Better Intellisense
- Waaay shorter build times
- Its Java
- Only references (nullables)
    - No copies or move constructors
- No lvalues and rvalues
- Readable
- One file per class
- Object types
- Switch on strings

## Cons

- Only Object Oriented Programming (OOPs all objects)
- Its's Java
- No operation overloading
- Only heap allocations
